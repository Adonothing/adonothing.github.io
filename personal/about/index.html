关于本人📃

  

  • 昵称：Adunas🍀 阿杜那斯🍀
  • 性别：♂
  • 生日：199X.08.05
  • 身份：珞珈山职业技术学校在读研究生
  • 地点：南方某个城市
  • 邮箱：2313047741@qq.com
  • 擅长：很难说擅长什么，涉猎广泛，都是皮毛
  • 为什么要起这个名字?
    1. 我需要取一个好记的英文名，和对应的中文名，于是有了故事。
    2. 曾在高中时偶然读过一本希腊神话故事书，里面有个美男子的名字叫 阿杜那斯。诚然这本书只是草草翻了几页，英文名字也是我杜撰的。
    3. 中文含义：姓杜那小子。那斯 又符合西方中文译名的味道，如 斯基。
    4. 名字是缘分、赞扬、打趣的组合，是姓氏和神话的组合，更是中西文化的交融。
  • 优点：
    1. 独一无二的真诚，总是想把最好的、最珍贵的东西与你分享，请你对我同样真诚。
    2. 世间少有的善良，希望你能开心，希望明天会更好，愿意为你不计回报地付出。
    3. 漫漫绵长的温柔，尊重你的选择，认真倾听你的心事，给出我心中最好的建议。
    4. 适度合理的幽默，喜欢开玩笑，但从不过分，希望逗你开心。
    5. 活跃跳脱的思维，跟得上节奏，理解得了新梗，总能提出好点子，有多重视角。
    6. 点燃大海的激情，热爱生活，热爱科技，想到明天会更好就会充满热情
  • 缺点：
    1. 不自觉陷入自卑，总是觉得事情做的不好，达不到想要的效果，没脸面对别人。
    2. 难坚持，真的很难，很难持续地把一件事做完、做好。
    3. 记忆力差，典型的理解记忆型，没有规律的、没理解逻辑的很难长期记忆，如单词。
    4. 间歇性身体疼痛，想挺直腰板，很难，偶尔会觉得头痛、脖子疼、腰疼。
    5. 慢，做事慢，重复或者机械劳动，和其他人相比，没有任何优势。

碎碎念

已经很棒了

  敢于学习做这个博客，已经很棒了，能无私分享一点有用的东西已经很棒了，没有必要苛责自己，已经很棒了。

我适合什么样的职业

  我适合什么样的职业呢？

  1. 算法工程师吗？我的代码能力并不行，我的记忆能力并不突出。
  2. 客户经理吗？我并不是一个外向的人，厚脸皮和高情商我并没有。
  3. 硬件工程师？非科班出生，能力不足。
  4. 博士、教授吗？我读论文并不容易，也不喜欢写论文，论文注水的风气也令人反感。作家？自以为是矫揉造作，从来也没有真正写过小说。
  5. 公务员？我受得了吗，刚正不阿，不会送礼，这么多年的学习技能得不到使用，一眼看到尽头，这是我想要的？
  6. 国企央企？稳定而腐朽，和私企不同，可能没有生存压力，所以风气陈旧？
  7. 创业吗？钱从哪儿来，你能做出什么产品，谁会买你的，你知道有多累吗？
  8. 回家种地？你脱得掉长衫吗？你舍得吗？你身体能做体力活吗？

我到底适合什么？什么东西适合我？

看看性格

缺点

  1. 看似视金钱为粪土，实则爱财如命，小气自私。
  2. 看似大公无私，实则小肚鸡肠，斤斤计较。
  3. 看似充满理想，实则无所事事，一事无成。
  4. 看似干劲十足，实则备受打击，动力不足。
  5. 想让别人看到，喜欢装逼，实则狐假虎威，叶公好龙。
  6. 内耗内耗，极度地内耗，担心这个做不好，那个做不好，担心惹这个不开心，惹那个不开心，小心翼翼，谨小慎微。
  7. 表面服从，心中反抗。觉得世界上没有人比自己更聪明，想自己决定，做自己地事情。然而事实是，自己做得更加一塌糊涂，难以坚持，不切实际。
  8. 喜欢的东西都拿不到，是彻彻底底的一个失败者。和别人在一起并不能开心，自己也没有能力满足自己的需求和欲望。
  9. 胆小害怕，害怕出糗。又极度的渴望在别人面前展示。
  10. 脑子里会对一些美好的，未能实现的，反复幻想。

优点

  1. 最佳副手，最佳助手，最佳助攻手。因为性格是 infj，具有敏锐的情感和需求观察力，能够迅速做出判断和调节，调整角色的定位，给予最好的辅助和支持、建议，但是要我独自一个人冲在前面，确实难受；要是主攻守，完全不信任我，也非常难受。
  2. 最佳操盘手，有感知危险的能力，有大局观，多视角的能力，超乎常人的想象力，不是一个顺序思维，是一个强跳跃思维，联想能力超绝。
  3. 顶级思想家，完备的自我完善体系，自我调节体系。
  4. 顶级的控制者，强大的幕后控场者，分析局势，控制局势，有着自己独一无二的能力
  5. 顶级的军事家，谋划、谋略都是符合局势，独一无二，精确对点的。

总结

  1. 对重复劳动、体力劳动没有优势，没有特点，干不过任何人。
  2. 坐不住，闲不住，对按部就班的事情做不了，需要一些新鲜感和激情。
  3. 带动氛围的能力还是很强，感染力还是很强。
  4. 正义化身，光明将最终战胜黑暗，所以一定会成功。

扬长避短

  总结就是我很强，但是一个人不够，我有战略目光，但小事难以坚持下来，有谋略，但是没有钱。我需要当领导者和先行者，但是又不够独，管理又狠不下心，天生最强二把手。

左脚踩右脚上天

  如果是二把手的话，这将永远是缺陷。人生是不可能指望别人的，别人不是你肚子里的蛔虫，更不可能和你完美互补。你很难遇到知音，就算遇到一次，第二次就不容易遇到了。如果遇不到就 g 了。

  如果你真的是天生最强辅助的话，首先要辅助的人就是自己，要成就的人就是自己，只有自己才是自己肚子里的蛔虫。
